# Resources

NASA source links, reference databases, and external resources for cislunar logistics research.

## Directory Structure

This directory provides organized access to:

- **NTRS Records** - Complete index of relevant NASA Technical Reports
- **Mission Pages** - Official NASA program and mission documentation
- **External Resources** - Academic papers, industry standards, reference databases
- **Reference Tools** - Orbital mechanics calculators, databases, lookup tools

## NASA Technical Reports Server (NTRS)

Primary resource for NASA publications: https://ntrs.nasa.gov

### Search Tips

- Use keywords: "cislunar," "gateway," "NRHO," "logistics"
- Filter by organization: NASA centers (MSFC, JSC, GRC, etc.)
- Filter by date for latest research
- Use NTRS ID for direct access: `https://ntrs.nasa.gov/api/citations/[ID]`

## Creating Resource Files

1. **Verify all NASA sources** through NTRS or official NASA pages
2. **Include direct links** to NTRS records
3. **Add NTRS IDs** when available for stable references
4. **Note publication dates** for time-sensitive information
5. **Organize by topic** for easy discovery
6. **Add brief relevance notes** explaining importance

## Resource Standards

All resources should include:

- [ ] Verified NASA source or official link
- [ ] Direct URL (preferably NTRS)
- [ ] Publication/access date
- [ ] Brief description of content
- [ ] Relevance to cislunar logistics
- [ ] Organization/NASA center attribution

## Updating Resources

- **Check links regularly** - ensure URLs remain active
- **Note when documents update** - track revisions
- **Add new resources** as NASA releases new materials
- **Archive obsolete materials** - note if superseded
- **Date all updates** - track when resource was verified

## External Resource Guidelines

Include non-NASA resources when:

- They provide important context or background
- They cite/reference NASA sources
- They help explain NASA concepts
- They are academically rigorous

Note: Primary citations should always be NASA sources.

## Access and Usage

When using resources from this directory:

1. Always cite the original NASA source
2. Use the NTRS link when available (more stable than direct URLs)
3. Document the date you accessed the resource
4. Note any important context or limitations
5. Link back to this resource collection for reference

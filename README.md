# Cislunar Logistics Network

A NASA-sourced educational research framework for studying logistics, transfer architecture, communications, navigation, and infrastructure across the Earth–Moon system.

## Purpose

This repository organizes research on cislunar logistics networks using NASA publications, NASA mission documentation, and NASA Technical Reports Server (NTRS) records as the primary citation base.

The project is designed for educational use, structured research, and long-term expansion inside the GGTC ecosystem.

## Scope

This repository covers:

- Cislunar region definition
- Orbital architecture
- Earth-to-NRHO transfer architecture
- Gateway and Near Rectilinear Halo Orbit (NRHO)
- Communications and navigation systems
- Logistics node mapping
- Energy and propulsion systems
- NASA source tracking
- Diagram and image metadata management

## Source Standard

Primary citations must come from NASA or NASA-hosted technical documentation.

Accepted source classes:

1. NASA official mission pages
2. NASA Technical Reports Server (NTRS)
3. NASA program documentation
4. NASA-authored conference papers or technical reports

Excluded by default:

- Wikipedia
- Unverified aggregators
- Unsourced commercial claims
- Non-authoritative summaries

## Integrity Rule

No unsupported operational claims are permitted.

Undefined areas remain intentionally undefined until supported by NASA documentation.

## Extended Description

This repository provides a structured, NASA-referenced research framework for analyzing and documenting cislunar logistics networks. It organizes the movement of crew, cargo, data, and energy across the Earth–Moon system through modular components including transfer architecture, Near Rectilinear Halo Orbit (NRHO) operations, Gateway as a logistics node, communications (DSN and LunaNet), and system-wide node mapping.

All technical content is grounded in NASA publications and the NASA Technical Reports Server (NTRS), with strict source-to-claim traceability. Undefined areas remain intentionally open until supported by verified documentation.

Developed as part of the GGTC.info ecosystem for educational use, long-term research, and structured systems analysis.
